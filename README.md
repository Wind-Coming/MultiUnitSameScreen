# MultiUnitSameScreen
 unity ecs 早期时候的版本
 当时真正做slg项目，看到很多slg项目在同屏很多单位时都会出现卡顿的情况，然后看到Unity官方宣传的ECS后，觉着太牛，并且感觉ECS特别适合slg项目，所以做了一个极限渲染单位的测试，改了一下底层渲染的合批方式来优化性能，最终在三星s8测试，同屏能达到十万单位并保持60帧。
 
 unity version:2019.4.19  
 entities version:0.11.1  
 jobs version:0.2.10
